Carnet de mon véhicule

Cette application a été conçue dans le but de faciliter le suivi des actions réalisées sur votre véhicule. 

Avec cette application mobile, vous pouvez enregistrer les voitures qui sont à votre disposition. 

Une fois qu'une voiture est enregistrée, vous avez la possibilité de lui ajouter les informations sur le propriétaire, les pannes, les entretiens, les visites techniques et la carte grise. 

Vous pouvez également en cas de vente de la voiture, transférer l'ensemble des données collectées au nouvel acquéreur afin qu'il sache ce qui a été fait par le passé sur le véhicule. 

Vous avez afin la possibilité de changer le thème, la taille de police ou encore la langue de l'application afin qu'elle soit plus conviviale pour vous.

Cette application, nous l'espérons, vous facilitera la vie dans la gestion de votre véhicule. 

Nous restons à l'écoute de vos commentaires et/ou suggestions pour une amélioration plus ergonomique. 

Merci pour votre soutien.
